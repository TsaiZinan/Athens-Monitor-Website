# CIVITAS 古典社会模拟

《CIVITAS 古典社会模拟》是一款非营利的开放的古典社会模拟网页游戏。它基于假想中的西汉—罗马时代，旨在以最简洁与灵活的方式，在抽象与实际、真实性与可操作性中寻求平衡[1]。

## 历史
CIVITAS 曾是一个每天有数万人共同经营的模拟城市，并由唯一开发者 “CIVITAS” 进行持续地更新。

然而现在，CIVITAS 已经不再继续获得更新，但仍有部分模拟居民参与其中。
